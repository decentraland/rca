# March 4th, 2026 - Avatar creation flow hung and login failures due to corrupt identity

|                        |               |
| ---------------------: | :------------ |
|  **Reported on Slack** | March-04-2026 |
| **Solution Completed** | March-04-2026 |

## What happened?

Users experienced a hung state in the avatar creation flow and the ones that didn't went through the avatar creation flow were unable to log in to the Explorer or in other sites due to a corrupt identity. The auth-site was released with a defect that caused these issues.

## Why did it happen?

A communication issue led to a release of the auth-site being deployed to production without going through proper QA testing. The untested release introduced two bugs, one that corrupted user identities due to the replacement of the Ethers library to reduce the bundle size and a crypto polyfill issue which prevented users from completing the avatar creation flow and completing any log in.

## Issue Timeline - UTC

1. 2026-03-04 14:00:00 - Incident reported
2. 2026-03-04 15:46:00 - Incident resolved

## Impact

- **Severity**: SEV-1
- Users were unable to complete the avatar creation flow, which appeared hung.
- Users were unable to log in to the Explorer due to a corrupt identity.

## Solution

The defective auth-site release was rolled back to restore normal functionality.
