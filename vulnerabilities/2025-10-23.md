# October 23, 2025 - GitHub Actions Script Injection

|                          |               |
| -----------------------: | :------------ |
| **Reported on Immunefi** | October-23-2025 |
|           **Mitigation** | October-23-2025 |
|   **Solution Completed** | October-23-2025 |

## Description

A new Github action created for the creator-hub was vulnerable to script injection.

## Severity

Websites & Applications - Critical.

## Solution

Remove any echo and use envs for the jobs runners in the [action](https://github.com/decentraland/creator-hub/blob/main/.github/workflows/ai-review.yml) .
