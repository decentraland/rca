# November 18th, 2025 - Cloudflare Global Network experiencing issues

## What happened?

Cloudflare's global network experienced widespread issues causing 500 errors across their services. The Decentraland Catalyst Network was mostly impacted by this incident, resulting in service disruptions for users.

## Why did it happen?

This was an external incident caused by Cloudflare's infrastructure experiencing problems. The root cause was on Cloudflare's side, not within Decentraland's systems. Full details available at: https://www.cloudflarestatus.com/incidents/8gmgl950y3h7

## Issue Timeline - UTC

1. 2025-11-18 12:22:00 - Incident reported (DCL-2080)
2. 2025-11-18 13:35 - Cloudflare support team continues working on restoring service. Decentraland Catalyst Network is mostly impacted by this issue.
3. 2025-11-18 14:42 - Cloudflare support team implemented a fix and is monitoring for any further issues.
4. 2025-11-18 17:44 - Cloudflare services are currently operating normally. No longer observing elevated errors or latency across the network.
5. 2025-11-18 18:03:00 - Incident resolved

## Impact

- **Severity**: SEV-2
- Widespread 500 errors affecting Decentraland services
- Decentraland Catalyst Network mostly impacted
- Users experienced service disruptions and inability to access certain features

## Solution

Cloudflare's engineering team resolved the issue on their end. The incident was external to Decentraland's infrastructure, so no internal changes were required. Cloudflare implemented a fix and monitored the platform, confirming services were operating normally by 17:44 UTC. It was considered safe to re-enable any Cloudflare services that were temporarily disabled during the incident.
